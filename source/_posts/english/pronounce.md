### 母音

### 五个规则

1. 词尾 `e` 不发音，前面的母音发元音（字母怎么读就怎么发音）
{% note warning %}
name 单音节单词末尾的e 不发音，a 发本身音a
take
ride
home
fine
{% endnote %}

2. 元音 + `r`
{% note success %}
- 单词开头/靠前，首次出现，发 `/ə/`
work
bird

- 单词尾出现，发 `/ər/`
worker
doctor
{% endnote %}

3. `oo` 发 `/ʊ/` 或 `/u:/`
{% note info %}
book `/bʊk/`
good `/gʊd/`
room `/ru:m/`
{% endnote %}

4. au, ou, aw, ow 发 `aʊ`
{% note danger %}
house
hour
how
{% endnote %}

5. 单独发音
|  字母   | 音标  |
|  ----  | ----  |
| th  | `/θ/`斯 `/ð/` di |
| ch  | `/tʃ/` 吃 |
| sh  | `/ʃ/` 失 |
| ph  | `/f/` 夫 |
| ng  | `/ŋ/` 恩 `/ŋg/` 恩哥 如 english `/'iŋ.gliʃ/` |
| nk  | `/ŋk/` 恩克 |

## 音节

|  重读闭音节   | 重读开音节  |
|  ----  | ----  |
| rap  | rape |
| bat  | bate |
| rop  | rope |
| kit  | kite |
| bus  | abuse |
| egg  | be |

### 重读开音节
- 元音 + 辅音 + e（不发音）
- 辅音 + 元音
{% note warning %}
元音发发音名称音，字母本身的读音
{% endnote %}

### 重读闭音节
- 元音 + 辅音
{% note warning %}
元音发短元音
{% endnote %}
