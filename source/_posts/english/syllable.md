## 音节表
由元音字母搭配元音或者辅音组成的音节
### 元音字母
{% label primary @a %}
{% label info @e %}

### 长元音
- 元音 + 元音
```sh
a: aa ae ai ao au
e: ea ee ei eo eu
i: ia ie ii io iu
o: oa oe oi oo ou
u: ua ue ui uo uu
```
- 元音 + 辅音 (`r`容`y`易`w`忘)
```sh
-e: a-e e-e i-e o-e u-e
re: are ere ire ore ure
w: aw ew iw ow uw
y: ay ey iy oy uy
r: ar er ir or ur
```

## 找音节
{% note success %}
1. 从后往前找元音
2. 元音前面切辅音
3. 从前往后读
{% endnote %}
### 拆解例子
- congratulation -> c`o`ng / r`a` / t`u` / l`a` / t`io`n
- parent -> p`a` / r`e`nt
- miracle -> m`i` / r`a` / c`le`

### 其他辅音

- qu `/kw/`
{% note info %}

{% endnote %}

- gu
  1. 出现在词首有元音 `/g/` {% label primary @guide %} {% label primary @guess %}
  2. 出现在词中，没有元音音节，`u`要发元音 {% label primary @August %} {% label primary @regular %}
  3. 出现在词中，有元音 `/gw/`  {% label primary @penguin %} {% label primary @language %}

- wr- {% label danger @w不发音 %} `/r/`
{% note success %}
write wrist wrong
{% endnote %}

- kn- {% label danger @k不发音 %} `/n/`
{% note success %}
know knock kness
{% endnote %}

- wh-
{% note success %}
wh 遇到 o，发 `/h/`, 其余发 `/w/`
{% endnote %}
  1. h不发音，`what` `why` `where`
  2. w不发音, `who` `whose` `whole`


### 辅音串
{% note info %}
为了更好地记住单词，两个字母都发音，2个音节
{% endnote %}

### L 串
- `bl` blue black blank blouse
- `cl` class close clean cloud
- `fl` flat flag float flower
- `gl` glass glue glad globe
- `pl` plan plus place please
- `sl` sleep slap slide slice

### R 串
- `br` blead brave bright bridge
- `cr` cry cross cream crayon
- `fr` fry free from frog
- `gr` grass grade green ground
- `pr` pray price print proud
- `thr` three thrive threat through

### S 串
- `sm` smile small smart
- `sn` snow snake snail
- `sw` sweet swing

### W 串
- `tw` twelve twins twig
- `dw` dwell dwarf
- `qu` quick queen question
- `-gu-` language penguin
